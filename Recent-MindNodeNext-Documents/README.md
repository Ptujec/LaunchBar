# LaunchBar Actions: Recent MindNode Next Documents

LaunchBar has built-in support for accessing [MindNode Next](https://mindnode.com/next) documents since version 6.22.

This custom action adds a few more features. You can paste a link to a selected document by holding `shift`, or in Markdown format including the title by holding `option` while pressing `return`.  
You can even create a `.mindnodealias` by holding `control`. The alias will be created in the directory of the current Finder window (with a fallback to the desktop).

## Limitations 

As MindNode Next documents are not stored as files but in a database, there are some things you cannot do, e.g. renaming a selected document or revealing it in the Finder. That's true of the built-in version as well.
